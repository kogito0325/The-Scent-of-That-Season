1. 수정된 변경사항은 다음 내용만 업데이트 한다.
- .vs
- Assets
- Documentation
- Logs
- obj
- Packages
- ProjectSettings
- 기타 낱개 파일들

2. 다음 내용은 절대 깃허브에 업데이트 하지 않고 git ignore에 추가한다.
- Library
- UserSettings

3. 풀 받은 프로젝트를 수정 할 땐 다음의 순서를 취한다.
- 프로젝트를 연다.
- 상단 Assets 탭에서 Reimport All 하여 애셋을 정상화 한다.
- 상단 Help 탭에서 Reset Packages to defaults 하여 프로젝트를 정상화 한다.
- 수정을 진행한다.

4. 기타 문제 사항이 생기면 진행을 멈추고 팀원에게 연락을 취한다. //중요함
